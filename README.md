# scheduler

a tool for managing daily tasks in linux console

## Getting started

Run `install.sh` to install required tools. This script will also set UTF-8 env.

```sh
git clone https://gitlab.com/pompelmous429/scheduler.git
cd scheduler
chmod +x install.sh
./install.sh [ubuntu|mac]
make
./scheduler
```
